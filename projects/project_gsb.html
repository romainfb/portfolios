    Le projet consiste à créer une application mobile pour les visiteurs médicaux de GSB afin de leur permettre d'accéder à des informations sur les médecins depuis leur tablette android en mode connecté ou déconnecté. Lorsqu'ils auront accès à internet, ils pourront utiliser des services web pour obtenir ces informations.
