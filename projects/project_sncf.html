    Ce projet consiste en la création d'un bot Twitter qui publie tous les matins à 9h des informations sur les gares et sur les objets perdus dans celles-ci. Ce bot utilise l'API de Twitter pour collecter les données en temps réel et les afficher sur le compte Twitter associé. Les utilisateurs peuvent ainsi avoir accès à des informations fiables et à jour sur les objets perdus dans les gares, les aidant ainsi à retrouver leurs objets perdus. La publication régulière de ces informations ajoute une valeur considérable pour les utilisateurs de Twitter.
